Digital Security Ethics, and Privacy

Group assignment completed in the class "ITEC 120: Introduction to Computer Hardware." Discussing security issues faced through the internet, such as malware and cyberattacks, and how easily an individual can be targeted. Points were also presented for implementation on how to solve these issues, such as setting a strong password, using biometric scans, muti-factor authentication, and more. Internet security can be a serious issue, and doing this assignment was a great lesson for an IT user.
