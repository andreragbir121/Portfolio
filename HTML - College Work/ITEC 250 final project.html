Hybrid SD-WAN topology

Final Project for the class "ITEC 250: Computer Network and Architecture Protocol" , concluded within a group of five with the intention of designing a topology for a network. My group implemented a hybrid software-defined wide area network (SD-WAN hybrid topology), which can be defined in summary as a technology that uses a software-based approach to manage and optimize wide area networks (WANs) more effectively than traditional hardware-centric ones. The SD-WAN design combined hubs and spokes and a partial mesh topology.
